Structure, Phrasing and Display

What is the difference between structural (block) elements and phrasing (inline) elements in HTML? Structural elements are tags that helps the engines interpret and categorize the codes more efficiently. They help determine the general shape of the codes in a more organized and legible way, making it easier for both coders and engines by using less div tags. Phrasing elements are tags that go in between the lines, that defines the blocked text. Some examples of phrasing elements are 'b', 'em', and 'abbr'.

What are the different settings for the display property in CSS, and how it can that be used to change the way elements are displayed or to create layout on the page? Display properties help structure the contents on the screen. Some settings for display property in CSS are inline, grid, and flex. These property values ease content organization by boxing them and providing specific values to place those boxes.

Box Model

CSS box model is a model of multiple layers of HTML elements; the most inner part is the content itself, then the padding, border, and margin at the very edge. This model is color coded and it helps coders see what element is taking up how much space in the webpage.

Box model can be changed by the box-sizing property by manipulating the width, height, padding, and margin of an HTML element.

Background Images

The images that are inside 'img' tag will be displayed at the specific location that it has been written on, whereas background image would appear behind that was assigned in. Another difference is that img tags are used in HTML file, whereas background-image is used in CSS file.
